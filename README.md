        Demographic and Health Surveys (DHS) Data Analysis – Kenya

    Project Overview
This project analyzes child nutrition and inequality in Kenya using Demographic and Health Survey (DHS) data.

The analysis focuses on key indicators such as:
    Stunting
    Wasting
    Underweight

    Objectives
  Estimate prevalence of child malnutrition indicators
  Compare urban vs rural differences
  Analyze inequality by wealth index

     Tools Used
 R Programming
 dplyr
 ggplot2

      Key Insights
 Higher malnutrition rates observed in rural areas
 Wealth inequality significantly affects nutrition outcomes
 Stunting is the most prevalent condition

     Project Files
 `analysis.R` → Main R script
 `report.pdf` → Full project report
