# DHS Child Malnutrition Analysis in Kenya

## Project Overview
This project analyzes child malnutrition in Kenya using Demographic and Health Survey (DHS) data.

The analysis focuses on:
- Stunting
- Wasting
- Underweight

## Inequality Dimensions Examined
- Residence (Urban vs Rural)
- Wealth Index
- Mother’s Education
- Sex of Child

## Methods Used
- Data cleaning
- Variable recoding
- Weighted prevalence analysis
- Inequality analysis
- Data visualization

## Tools and Packages
- R
- R Markdown
- dplyr
- tidyr
- ggplot2
- haven

## Outputs
- R Markdown analytical report
- HTML report
- Graphical visualizations
